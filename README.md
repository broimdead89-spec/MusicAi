## 📌 О проекте

**MusicAI** — это веб-сервис нового поколения, который объединяет музыкальный каталог из YouTube Music с возможностями искусственного интеллекта. Система анализирует ваши музыкальные предпочтения и позволяет создавать персональные плейлисты через диалог с ИИ-агентом на естественном языке.

> 🎯 **Источник данных**: все треки и метаданные поступают исключительно из YouTube Music.

## ✨ Возможности

### 🤖 ИИ-агент для плейлистов
Просто напишите, что хотите услышать:
```
"Подбери плейлист для утренней пробежки, энергичный, без слов, около 30 минут"
"Хочу что-то меланхоличное в стиле инди для осеннего вечера"
"Лоуфай для работы, не более 40 минут"
```
Агент поймёт настроение, жанр, эпоху и соберёт плейлист из базы YouTube Music. Можно уточнять: *"добавь больше женских вокалов"*, *"сделай потише по темпу"*.

### 📊 Персональные рекомендации (ML)
- Анализ истории прослушиваний, лайков и скипов
- Похожие исполнители и жанры
- Учёт временных паттернов (утро/вечер, будни/выходные)
- Адаптация к смене музыкальных вкусов

### 🔍 Каталог треков
- Поиск по названию, артисту, жанру
- Фильтрация и сортировка
- Обложки, длительность, метаданные

### ❤️ Моя музыка
- Лайки и избранное
- История прослушиваний
- Персональные плейлисты

## 🚀 Быстрый старт

### Требования
- Python 3.10+
- Node.js 18+
- PostgreSQL 14+ (или SQLite для разработки)

### Установка

```bash
# 1. Клонировать репозиторий
git clone https://github.com/ВАШ_НИК/MusicAI.git
cd MusicAI

# 2. Настроить бэкенд
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env       # Заполнить переменные окружения

# 3. Запустить бэкенд
uvicorn main:app --reload

# 4. Настроить фронтенд
cd ../frontend
npm install
npm run dev
```

### Переменные окружения (.env)

```env
# База данных
DATABASE_URL=postgresql://user:password@localhost:5432/musicai

# YouTube Music (ytmusicapi)
YTM_COOKIE=ваши_куки_из_браузера

# LLM (OpenAI / локальная модель)
OPENAI_API_KEY=sk-...
LLM_MODEL=gpt-4o-mini

# JWT
SECRET_KEY=ваш_секретный_ключ
```
